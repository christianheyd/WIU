Project Description

This project is for a class on Full Sail Online called "Web Interface and Usability." The objective of this class is to give the students a greater understanding of how to create proper layouts that follow rules of usability and interactivity. This is done mainly through following design patterns. This page has been created to link users to my wireframes I've created for the class for usability testing. I encourage anyone who stumbles across this to try one out!
